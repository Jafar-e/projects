Frontend разработка

Введение во Frontend разработку

Frontend разработка отвечает за создание пользовательской части веб-сайта или веб-приложения. Она включает в себя работу с HTML, CSS и JavaScript.

Языки Frontend разработки

Инструменты Frontend разработки

  1. Visual Studio Code – мощный редактор кода c поддержкой множества языков и плагинов;
  2. React – библиотека для создания пользовательских интерфейсов;
  3. Sass – пререпроцессор CSS c расширенными возможностями;
  4. Webpack – сборщик, используется для добавления интерактивности на веб-страницу, такой как анимации, обработка событий и динамическое обновление содержимого.
