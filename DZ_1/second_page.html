Ключевые технологии:

HTML
  • Структура: Использует теги для определения различных элементов
    (заголовки, абзацы, изображения, ссылки и т.д.).
  • Атрибуты: Дополнительные параметры тегов
    (например, src для <img>, href для <a>).
  • Документная модель: HTML-документ представляет собой дерево элементов.
  • Версии: Текущая версия HTML5, включает новые теги и улучшенные функциональные возможности
    (например, <video>, <audio>, <canvas>).
  • Совместимость: Поддерживается всеми основными веб-браузерами.
  • Использование: Основной язык для создания и структурирования веб-страниц и веб-приложений.

CSS
  • CSS (Cascading Style Sheets): Каскадные таблицы стилей, используются для описания внешнего вида и форматирования веб-страниц.
  • Цель: Отделение содержимого (HTML) от его представления (CSS).
  • Структура: Состоит из правил (селекторов и деклараций).
    • Селектор: Определяет, к каким HTML-элементам применяется стиль.
    • Декларация: Состоит из свойства и значения, определяющих стиль
      (например, color: red;).
  • Способы включения CSS:
    • Внешние таблицы стилей: Подключаются с помощью тега <link> в <head>
      (например, <link rel="stylesheet" href="styles.css">).
    • Внутренние таблицы стилей: Встраиваются в HTML-документ с помощью тега <style> в <head>.
    • Встроенные стили: Определяются непосредственно в атрибутах HTML-элементов
      (например, <div style="color: red;">).
  • Основные свойства:
    • Цвет и фон: color, background-color, background-image.
    • Текст: font-family, font-size, text-align, line-height.
    • Блоки: margin, padding, border, width, height.
    • Расположение: position, display, float, flex.
  • Каскадирование: Принцип, по которому стили применяются в зависимости от их источника и специфичности
    (внешний, внутренний, встроенный).
  • Наследование: Некоторые свойства наследуются от родительских элементов
    (например, color, font-family).
  • Специфичность: Приоритет стилей определяется по уровню специфичности селектора.
  • Медиазапросы: Позволяют адаптировать стили под разные устройства и размеры экранов
    (например, @media (max-width: 600px) { ... }).
  • Версии: Текущая версия CSS3, включает новые возможности и улучшенные свойства
    (например, градиенты, тени, анимации).
  • Поддержка: Поддерживается всеми основными веб-браузерами.

JavaScript
  • JavaScript: Скриптовый язык программирования, используемый для создания динамического и интерактивного веб-контента.
  • Типы данных: Основные типы данных включают Number, String, Boolean, Object, Null, Undefined, Symbol, и BigInt.
  • Переменные: Используются для хранения данных; объявляются с помощью var, let, и const.
  • Функции: Блоки кода, которые выполняют определенные задачи и могут быть вызваны при необходимости.
    • Объявление функции: function myFunction() { /* код */ }
    • Стрелочные функции: const myFunction = () => { /* код */ }
  • Объекты и массивы:
    • Объекты: Коллекции пар ключ-значение, например, const obj = { key: 'value' };
    • Массивы: Упорядоченные списки элементов, например, const arr = [1, 2, 3];
  • Условия: if, else if, else для выполнения кода на основе условий.
  • Циклы: for, while, do...while для повторения выполнения кода.
  • DOM (Document Object Model): Интерфейс для взаимодействия с HTML и CSS, позволяет изменять структуру, стиль и содержание веб-страниц.
    • Доступ к элементам: document.getElementById('id'), document.querySelector('.class')
    • Изменение элементов: element.innerHTML, element.style.property
  • События: Механизм для реакции на действия пользователя или изменения состояния
    (например, click, mouseover).
    • Обработчики событий: element.addEventListener('event', function)
  • Асинхронное программирование:
    • Колбэки: Функции, передаваемые в другие функции для выполнения позже.
    • Промисы: Объекты, представляющие завершение или неудачу асинхронной операции
      (например, fetch API).
    • async/await: Синтаксический сахар для работы с промисами, делает код более читаемым.
  • JSON (JavaScript Object Notation): Формат обмена данными, используемый для передачи данных между сервером и клиентом.
  • Библиотеки и фреймворки: Популярные инструменты для упрощения разработки
    (например, jQuery, React, Angular, Vue.js).
  • Поддержка: JavaScript поддерживается всеми основными веб-браузерами и часто используется в комбинации с HTML и CSS для создания веб-приложений.
